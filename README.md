# ExpressJS OOP Template

Dự án này là template ExpressJS theo chuẩn OOP sử dụng TypeScript. Cấu trúc dự án chia thành các class cho controller, service, và routes.

## Khởi động dự án

```bash
npm run dev
```

## Cấu trúc thư mục đề xuất
- src/
  - controllers/
  - services/
  - routes/
  - app.ts
